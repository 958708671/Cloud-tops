# Minecraft 服务器白名单管理系统

## 项目说明

这是一个 Minecraft 服务器的白名单管理系统源代码，仅用于服务器内部管理使用。

## 系统功能

- **白名单申请**：玩家在线提交申请
- **审核管理**：管理员审核白名单申请
- **黑名单管理**：管理违规玩家
- **公告管理**：发布服务器公告
- **活动管理**：创建和管理服务器活动
- **投诉处理**：处理玩家投诉
- **管理员管理**：权限分级管理

## 技术栈

- Next.js 16
- React
- TypeScript
- PostgreSQL
- Vercel 部署

此项目仅供内部使用，不对外公开部署指南。
